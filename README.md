# Student Management System (Java)

## 📌 Project Description
This is a simple Student Management System developed using Java.  
The application runs in the console and allows users to manage student records easily.

Users can add, view, search, update, and delete student details.

## 🚀 Features
- Add new student
- View all students
- Search student by ID
- Update student details
- Delete student record
- Menu-driven console program

## 🛠 Technologies Used
- Java
- ArrayList
- Scanner (User Input)
- VS Code / Any Java IDE

## 📂 Project Structure
StudentApp.java

Inside this file:
- Student class → Stores student data (ID, Name, Age)
- StudentApp class → Contains the main program logic

## ▶️ How to Run the Program

1. Open the project folder in VS Code  
2. Compile the Java file

javac StudentApp.java

3. Run the program

java StudentApp
